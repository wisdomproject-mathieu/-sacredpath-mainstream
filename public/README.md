# Intimacy Weather Image Assets

These 12 images were extracted from the Shiva & Shakti weather board and named with stable slugs for Codex/frontend use.

Use `archetype` as `shiva` or `shakti`; use `weather` as one of:
`stormy`, `foggy`, `frozen`, `warm`, `electric`, `sunny`.

Recommended path after copying into the app:
`public/assets/weather/<slug>.png`
